This repository provides **prepared starting files for Project Diablo 2 map making**.

The intention of these files is to give you an **already set up, mostly empty map** along with all necessary edited text files, so you can immediately start creating and testing maps without having to build everything from scratch.

These files allow you to:
- Start with an empty map template
- Modify and edit map layouts easily
- Test graphics, objects, and monsters **ingame**
- Compare how assets look in the editor vs. ingame (since visuals can differ)

## ðŸ—ºï¸ How to Enter the Map

1. Buy an **Identify Scroll** from any NPC.
2. Put the scroll into the **Horadric Cube**.
3. Transmute it.

This will create a **small teddy**.

4. Go to **Act 5 on Hell difficulty**.
5. Right-click the teddy (just like opening a regular map).
6. Enter the map through the **red portal**.
